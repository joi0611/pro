# RAP LAB 静态网页

这是一个不需要后端的引导式说唱练习网页。四拍热身、歌词创作、自动分拍、结尾尾巴、合成节拍、麦克风录音、混音、回放和下载都在浏览器本地完成。

## 启动方式

麦克风功能要求网页运行在 `localhost` 或 HTTPS 环境中，请不要直接双击 `index.html` 后测试录音。

在本目录打开 PowerShell，运行：

```powershell
py -m http.server 8080
```

然后在 Chrome 或 Edge 打开：

```text
http://localhost:8080
```

第一次点击“开启麦克风”时，选择允许。建议戴耳机录制，避免扬声器里的节拍被麦克风再次收进去。

## 当前功能

- 首页与“今日录音棚”分离的两段式体验
- 四拍热身、事件跟读、句式填空、自由创作、完成 TRACK 五级升级路线
- 4 个“今日离谱事件”，点击事件后立即载入对应内容
- 每个事件自动生成 4 句初级跟读歌词，不需要再次选择关键词
- 每个事件自动匹配一个独立 Ad-lib 口头语尾巴
- 带 90 BPM 节拍灯的滚动歌词提词屏
- 逗号前后自动拆成两个独立提词画面
- “今天____突然____”等 4 组渐进句式支架
- 第一句事件关键词自动带入并锁定，不能误改
- 每个事件拥有独立的“给我灵感”词库，切换事件时不会串题
- 可选择的收尾句与逐空格灵感填充
- 事实、夸张、转折、收尾四步自由创作
- 重复、反转、夸张、对比、拟人五种“最强一句”技巧
- 自由创作继承并锁定前四句，只新增话题结尾与最强一句
- “一句话装进四拍”提供 4 套数字节奏型：`1 / 2 / 3 / 4`、`12 / 12 / 12 / 3`、`1234 / 1234 / 1234 / 12`、`1234 / 12 / 1234 / 12`
- 节拍训练器点击后直接起拍；歌词跟读和录音仍有 `3、2、1、GO` 倒计时
- 90 BPM 明亮电子 Hip-Hop Beat，包含稳定鼓组、大调合成器短音、环境铺底与空间混响
- 自动生成完整 6 BAR 最终 TRACK 与提词练习
- 麦克风声波、3 秒倒计时、最长 60 秒录音
- 人声与节拍本地混音、立即回放和文件保存
- 桌面、平板和手机响应式布局

录音不会上传。刷新页面后，尚未下载的录音会被清除。
